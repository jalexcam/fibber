# Security Policy

## Supported Versions

The following versions of `jalexcam/fibber` have active support:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | Jun 07th, 2026     |

## Reporting a Vulnerability

If you believe that you have found a security vulnerability, please send an email to security@fibberpackage.org. Ensure to include all details required to understand the severity of the issue.
